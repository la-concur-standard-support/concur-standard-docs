Web サービス

概要

  1. Concurforce 製品では、Web サービスは利用できません。

管理者は、[Web サービス] ページを使用して Concur プラットフォーム向けのパートナー アプリケーションを管理します。このページのリンクから、Concur で使用するパートナー アプリケーションを登録したり、パートナー アプリケーションを有効化して Concur データにアクセスしたりできます。

基盤

Concur プラットフォームの基盤は以下から構成されます:

  • Web サービスを使用するパートナー アプリケーションを登録するためのユーザー インターフェイス
  • 会社ごとにパートナー アプリケーションを有効化するためのユーザー インターフェイス
  • アプリケーションのアクセス レベルを決定する共通の認証および許可モデル
  1. いくつかの Web サービスは、Concur Professional/Premium 製品でのみ利用可能です。
  • [Web サービス] ページと利用可能な Web サービスの詳細については、http://developer.concur.com をご参照ください。
