会計ファイル出力構成

概要

設定ウィザードの [会計ファイル出力] ステップは、特定のお客様にご利用いただける機能です。QuickBooks を使用しているお客様は、出力設定を修正できません。

形式を選択

[設定] の [形式を選択] ページでは、支払マネージャーでバッチを終了したときに受け取る取引出力ファイルの形式をカスタマイズできます。カスタム形式を使用するには、次のステップから成るウィザードを実行します。

  • 出力する文字データ: このステップでは、出力する文字データを構成しているフィールドと固定値を選択して、出力する文字データを構成できます。固定長、パディング文字、位置揃えを選択できます。
  • 全般設定: このステップでは、会計出力ファイルの形式全般を構成できます。ファイルに、引用符、列ヘッダー、または相殺入力が含まれるかどうかを選択できます。区切り文字、日付形式およびファイル拡張子を選択できます。
  • ヘッダー: このステップでは、結果をグループ化するために使用するフィールドを選択できます。結果をグループ化する場合、ヘッダー行に表示するフィールドを選択できます。
  • 詳細フィールド: このステップでは、各仕訳入力に表示するフィールドを選択し、フィールドの順序を修正できます。
  • サマリー/コピー: このページには、選択した構成のサマリーが表示され、ウィザードを完了する前に構成を確認できます。他の支払バッチを選択して、そこに現在の構成をコピーすることもできます。

この形式は、選択したタイプの以降のバッチすべてに適用されます。後で必要に応じて [設定] に戻り、変更を行うことができます。

例

会計出力ファイルをカスタマイズするとします。[オプション 2] を選択すると、ウィザードの残りのステップにアクセスしてカスタマイズを進められるようになります。

[会計ファイル出力 - ファイル形式] ページへのアクセス

  1. [会計ファイル出力 - ファイル形式] ページにアクセスするには:
  2. [管理] ＞ [設定] ＞ [経費] に進みます。
  3. [経費] ＞ [会計ファイル出力] をクリックします（左側メニュー）。[会計ファイル出力 - ファイル形式] ページの [ファイル形式] タブが表示されます。

会計出力ファイル形式の選択

  1. 会計出力ファイル形式を選択するには:
  2. [会計ファイル出力 - ファイル形式] ページで、目的のファイル形式オプションを選択します。
  3. [次へ] をクリックします。[オプション 1] を選択した場合、次の [設定] ステップに進みます。[オプション 2] を選択した場合、[会計ファイル出力] ウィザードが進みます。

出力する文字データを選択

[出力する文字データ] タブで、すべての支払バッチの出力する文字データ作成に使用するフィールドを追加し、その順序を調整できます。フィールドを組み合わせて使用して、元帳に必要な出力する文字データを作成します。固定値を出力する文字データに追加したり、位置揃え、パディング文字、各フィールドの長さを設定したりできます。

例

固定値（US）、従業員 ID、勘定科目コードで構成される出力文字データがあります。従業員 ID は、常に 10 桁にし、10 桁ない場合は先頭にゼロを付ける必要があります。US と入力して [固定値を追加] をクリックします。値「US」が右側のボックスに表示されます。[従業員 ID] をクリックして、[追加] をクリックします。右側のボックスに、固定値の後にフィールドが表示されます。[勘定科目コード] フィールドを選択して、[追加] をクリックします。右側のボックスに、従業員 ID の後にフィールドが表示されます。[従業員 ID] 行の [位置揃え] フィールドを選択して、[右] を選択します。[パディング文字] フィールドを選択して、[ゼロ] を選択します。[固定長] フィールドを選択して、10 と入力します。

[出力する文字データ] タブへのアクセス

  1. [出力する文字データ] タブにアクセスするには:
  2. [管理] ＞ [設定] ＞ [経費] に進みます。
  3. [経費] ＞ [会計ファイル出力] をクリックします（左側メニュー）。[会計ファイル出力 - ファイル形式] ページの [ファイル形式] タブが表示されます。
  4. [出力する文字データ]（タブ）をクリックします。

フィールドおよび出力する文字データの選択

  1. 出力する文字データのフィールドを選択するには:
  2. 左側のペインで 1 つまたは複数のフィールドを選択します。
  3. [追加] をクリックします。選択したフィールドが右側のペインに表示されます。

固定値の追加

  1. 出力する文字データに固定値を追加するには:
  2. テキスト フィールドに固定値を入力します。

  1. [固定値を追加] をクリックします。値が右側のペインに表示されます。

フィールド設定の更新

  1. フィールド設定を更新するには:
  2. フィールドの [位置揃え] 列で値を選択します。
  • 右詰めの場合はデータ値が右に移動し、先頭のスペース（値の左側）がパディング文字列で満たされます。
  • 左詰めの場合はデータたちが左に移動し、末尾のスペース（値の右側）がパディング文字列で満たされます。
  1. フィールドの [パディング文字] 列で値を選択します。
  2. フィールドの [固定長] 列で値を入力します。

フィールドの並べ替え

  1. フィールドを並べ替えるには:

[順序] 列で、移動するフィールドの矢印をクリックします。

出力する文字データからのフィールドの削除

  1. 出力する文字データからフィールドを削除するには:
  2. 右側のペインで目的のフィールドを選択します。

  1. [削除] をクリックします。フィールドが左側のペインに移動します。

全般設定

[全般設定] タブで、支払バッチに基づいて会計出力ファイル形式の全般設定を指定できます。相殺入力もファイルに追加できます。

例

コンマ区切りで、入力の前後に引用符を配置し、相殺入力がある従業員の会社小切手バッチの会計出力ファイルが必要であるとします。支払バッチを選択して、[ファイル内のテキスト フィールドおよびリスト フィールドの前後に引用符を配置する] チェック ボックスを選択します。[経費グループごとに総勘定元帳への相殺入力を作成する] チェック ボックスを選択し、相殺入力に使用する勘定科目コードを入力します。[ファイル内のフィールド区切文字を選択] フィールドで、コンマを選択します。

[全般設定] タブへのアクセス

  1. [全般設定] タブにアクセスするには:
  2. [管理] ＞ [設定] ＞ [経費] に進みます。
  3. [経費] ＞ [会計ファイル出力] をクリックします（左側メニュー）。[会計ファイル出力 - ファイル形式] ページの [ファイル形式] タブが表示されます。
  4. [設定]（タブ）をクリックします。

全般設定

  1. 全般設定を更新するには:
  2. [全般設定] タブで、設定する支払バッチを選択します。

  1. 以下のオプションの設定を入力します。

フィールド

説明

ファイル内のテキスト フィールドおよびリスト フィールドの前後に引用符を配置する

ファイル内のすべてのフィールドの前後に二重引用符を配置するには、このオプションを選択します。

注意: このオプションは、コンマ区切りファイルで、列の順序を乱さずにコンマがある値を維持したい場合に推奨されます。

ファイル最上部に説明用の列ラベルを含める

ヘッダー、詳細、およびサマリー フィールドのファイル最上部に列ラベルを含めるには、このオプションを選択します。

経費グループごとに総勘定元帳への相殺入力を作成する

経費のグループごとに相殺 GL 入力を作成するには、このオプションを選択します。相殺の値は、デビット入力の総計と同額の、符号が逆の値です。

フィードに相殺入力を生成する場合、相殺勘定科目コードを選択

前のチェック ボックスを選択した場合、相殺入力に使用するには、勘定科目コードの値を入力します。

ファイル内のフィールド区切文字を選択

会計出力ファイルでフィールドの区切りに使用する文字を選択します。

日付形式を選択

会計出力ファイルですべての日付に使用する形式を選択します。

ファイル拡張子を選択

会計出力ファイルの拡張子を選択します。

税詳細を別の行に表示

勘定抽出での税詳細の表示方法を選択するオプションです。

  • いいえ: 税詳細と経費を組み合わせて、会計出力ファイルの単一の行に表示します。
  • はい: 抽出データ内で取引ごとに少なくとも 2 行に税詳細を表示します。1 行は経費、もう 1 行は税詳細です。税詳細は、税の行にのみ表示されます。
  • 別の行と Expense の両方に表示: 抽出データ内で取引ごとに少なくとも 2 行に税詳細を表示します。1 行は経費、もう 1 行は税詳細です。税詳細は、経費と税の行両方に表示されます。

注意: 税詳細フィールドは、国の運用に関わらず表示されます。この機能は、VAT がある国向けのものですが、米国など、 VAT がない国のお客様にも表示されます。

  1. [次へ] をクリックします。

ヘッダー

[ヘッダー] タブで、各支払バッチの経費詳細行のグループ化に使用するフィールドを選択します。グループ化した行のヘッダーに表示するフィールドも、このステップで構成します。

例

出力する文字データ別に支払バッチ入力をグループ化し、ヘッダーに従業員 ID、金額、レポート名およびレポート提出日を表示するとします。目的の支払バッチ タイプを選択して、結果をグループ化する出力する文字データを選択します。目的のフィールドを一度につき 1 つ選択して [追加] をクリックします。

  • 使用可能なフィールドの定義については、ユーザーガイド「Expense: Concur Admin」をご参照ください。

[ヘッダー] タブへのアクセス

  1. [ヘッダー] タブにアクセスするには:
  2. [管理] ＞ [設定] ＞ [経費] に進みます。
  3. [経費] ＞ [会計ファイル出力] をクリックします（左側メニュー）。[会計ファイル出力 - ファイル形式] ページの [ファイル形式] タブが表示されます。
  4. [ヘッダー]（タブ）をクリックします。

グループ ヘッダーのプロパティの設定

  1. グループ ヘッダーのプロパティを設定するには:
  2. 修正する支払バッチを選択します。
  3. 経費詳細行のグループ化に使用するフィールドを選択します。
  4. ヘッダー行のフィールドを選択して、[追加] をクリックします。
  5. 必要に応じて、固定値を入力して [固定値を追加] をクリックします。
  6. 上下の矢印を使用してフィールドを並べ替えます。
  7. [次へ] をクリックします。

詳細フィールド

[詳細フィールド] タブで、詳細行に表示するフィールドを選択し、支払バッチごとに経費詳細レコードを構成できます。管理者は、固定値を追加して、フィールド順序を並べ替えることもできます。カスタマイズ可能なフィールドの標準セットが、Concur によって提供されます。

  1. [経費詳細フィールドを選択] ペインの [金額] フィールドを選択および設定している場合は、抽出を修正する際に、取引通貨（ユーザーが支出するときの通貨）と払戻通貨（ユーザーへの支払の時の通貨）が異なる可能性があることに注意してください。たとえば、ユーザーが英国にいるときに英国ポンドで支出し、払戻は米国ドルで行う場合です。

例

経費ごとに、従業員の姓名および承認金額を収集する必要があるとします。これらのフィールドを一度に 1 つずつ選択して [追加] をクリックし経費詳細フィールドに追加します。右側のペインの上下の矢印をクリックして、フィールドを並べ替えます。

[詳細フィールド] タブへのアクセス

  1. [詳細フィールド] タブにアクセスするには:
  2. [管理] ＞ [設定] ＞ [経費] に進みます。
  3. [経費] ＞ [会計ファイル出力] をクリックします（左側メニュー）。[会計ファイル出力 - ファイル形式] ページの [ファイル形式] タブが表示されます。
  4. [詳細フィールド]（タブ）をクリックします。

詳細フィールドの管理

詳細フィールドを追加、削除して、各経費取引に表示する情報をカスタマイズできます。フィールドの追加および削除に加えて、[置換] ボタンを使用して既定またはその他のフィールドをファイル出力から削除し、目的のフィールドを追加できます。

  1. 管理者は、一度に 1 つのフィールドを切り替えることができます。
  • 使用可能なフィールドの定義については、ユーザーガイド「Expense: Concur Admin」をご参照ください。

詳細フィールドの追加

  1. 詳細フィールドを追加するには:
  2. 設定する支払バッチを選択します。
  3. 目的のフィールドを選択して [追加] をクリックします。
  4. 必要に応じて、固定値を入力して [固定値を追加] をクリックします。
  5. 上下の矢印を使用してフィールドを並べ替えます。
  6. [次へ] をクリックします。

詳細フィールドの置換

  1. 詳細フィールドを置換するには:
  2. [経費詳細フィールドを選択] 列にあるフィールドと [詳細フィールドを順序付け] 列のフィールドをクリックします。
  3. [置換] をクリックします。[経費詳細フィールドを選択] リストで選択したフィールドが、[詳細フィールドを順序付け] リストの選択した位置に挿入されます。[詳細フィールドを順序付け] リストで選択したフィールドは、[経費詳細フィールドを選択] リストのアルファベット順に配置されます。

サマリー/コピーの構成

[サマリー/コピー] タブでは、選択した支払バッチの完全な構成を確認できます。任意のバッチの構成を、1 つ以上の別の支払バッチにコピーすることもできます。

例

すべての支払バッチを同じ会計出力ファイル形式にする必要があるとします。リストですべてのバッチを選択して [コピー] をクリックして、完了した構成を別のバッチにコピーします。

[サマリー/コピー] タブへのアクセス

  1. [サマリー/コピー] タブにアクセスするには:
  2. [管理] ＞ [設定] ＞ [経費] に進みます。
  3. [経費] ＞ [会計ファイル出力] をクリックします（左側メニュー）。[会計ファイル出力 - ファイル形式] ページの [ファイル形式] タブが表示されます。
  4. [サマリー/コピー]（タブ）をクリックします。

ファイル構成のコピーおよびエクスポート

  1. 会計出力ファイル設定をコピーするには:
  2. 目的の支払バッチを選択します。
  3. 設定のコピー先のバッチを選択します。

  1. [コピー] をクリックします。
